# Organizador de archivos de estudio 

Script para ordenar documentos de materias de estudio, clasificándolos en "teoría", "práctica" o "indefinidos"


## Cómo lo uso?
1. Clona este repo
2. Abrí la carpeta en la terminal y ejecutá `python organizador.py`
3. Copiá la ruta de la carpeta a ordenar y pegala cuando el programa te lo pida
